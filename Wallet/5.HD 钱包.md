BIP-39

在 JavaScript/TypeScript 中，`bip39` 是一个常用库，用于处理 **助记词（mnemonic）和种子（seed）之间的转换**，常用于钱包生成（与 BIP-32/BIP-44 搭配使用）。

| 方法名                            | 说明                |
| ------------------------------ | ----------------- |
| `generateMnemonic()`           | 随机生成助记词           |
| `validateMnemonic(mnemonic)`   | 校验助记词是否有效         |
| `mnemonicToSeed(mnemonic)`     | 将助记词转换为种子（Buffer） |
| `mnemonicToSeedSync(mnemonic)` | 同上（同步版本）          |
| `entropyToMnemonic(entropy)`   | 用给定熵生成助记词         |
| `mnemonicToEntropy(mnemonic)`  | 将助记词转换回熵          |
| `wordlists`                    | 内置支持的词表（如英文、中文等）  |

**熵 → 助记词 → 种子 → 私钥 → 地址**
