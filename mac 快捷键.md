**macOS**

- **全屏截图**：`Command (⌘) + Shift + 3`
    
- **选取区域截图**：`Command (⌘) + Shift + 4`（按住后用鼠标拖拽选取）
    
- **窗口截图**：`Command (⌘) + Shift + 4`，然后按一次 **空格** 键，再点击窗口
    
- **截图并直接复制到剪贴板**：在上述组合键的基础上多按 `Control`
    
- **截图工具栏**（可录屏）：`Command (⌘) + Shift + 5`

## 命令行编辑快捷键（光标和文本操作）

这些和 Linux/Unix 的 Bash、zsh 都通用：

- **Ctrl + A**：移动到行首（相当于 Home）
    
- **Ctrl + E**：移动到行尾（相当于 End）
    
- **Ctrl + U**：删除光标前的所有内容
    
- **Ctrl + K**：删除光标后的所有内容
    
- **Ctrl + W**：删除光标前的一个单词
    
- **Ctrl + Y**：粘贴上一次用 `Ctrl + U/K/W` 删除的内容
    
- **Ctrl + H**：退格（相当于 Backspace）
    
- **Ctrl + D**：删除光标处字符；在空行时退出终端
    
- **Ctrl + L**：清屏（相当于 `clear`）
    
- **Option + → / ←**：按单词移动光标
    
- **Ctrl + T**：交换光标前两个字符

## 终端窗口控制快捷键

- **Cmd + T**：新建标签页
    
- **Cmd + N**：新建窗口
    
- **Cmd + W**：关闭当前标签页/窗口
    
- **Cmd + ↑ / ↓**：向上/下翻页
    
- **Cmd + K**：清空屏幕内容（比 `clear` 更彻底）
    
- **Cmd + 数字**：切换到对应标签页
    
- **Cmd + Shift + [ / ]**：切换上/下一个标签页
    
- **Cmd + +/-**：放大/缩小字体
    
- **Cmd + 0**：恢复默认字体大小
    
- **Cmd + C**：复制选中文本 / 中断当前进程（取决于是否有选中内容）
    
- **Cmd + V**：粘贴
    
- **Cmd + F**：搜索终端输出

## 进程管理快捷键

- **Ctrl + C**：强制中断当前进程
    
- **Ctrl + Z**：挂起当前进程（放后台）
    
- **fg**：恢复前台运行
    
- **bg**：后台运行挂起的进程
