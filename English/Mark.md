- 。 句号 → period / full stop（英式更常说 full stop）
    
- ， 逗号 → comma
    
- ； 分号 → semicolon
    
- ： 冒号 → colon
    
- ？ 问号 → question mark
    
- ！ 感叹号 → exclamation mark / exclamation point（美式常说 point）
    

### 引号 / 括号 / 省略

- “ ” 双引号 → quotation marks / double quotation marks
    
- ‘ ’ 单引号 → single quotation marks
    
- ‘ 省略撇号（I'm 里的 ’）→ apostrophe
    
- （ ） 圆括号 → parentheses / round brackets
    
-  方括号 → square brackets / brackets
    
- { } 花括号 → curly brackets / curly braces / braces
    
- …… 省略号 → ellipsis  
    （单数：an ellipsis；复数：ellipses）
    

### 连接/分隔符

- - 连字符 → hyphen  
        （如 well-known）
        
- – 短破折号 → en dash  
    （常用表示范围：2019–2025）
    
- — 长破折号 → em dash  
    （句子里插一句话——像这样）
    
- 、 顿号 → pause mark / enumeration comma  
    _英文里基本不用这个符号，但可以这样解释_
    

### 引用/对话常见

- ： 冒号（如：他说：…）→ colon（同上）
    
- —— 破折号（中文的）→ em dash（roughly equivalent）
    

### 书写中常见的符号

- . 点 → dot / period  
    （在网址、缩写、小数点时常叫 dot）
    
- · 中点 → middle dot / interpunct  
    （人名分隔用的·，比如西语名）
    
- / 斜杠 → slash / forward slash
    
- \ 反斜杠 → backslash
    
- | 竖线 → vertical bar / pipe
    
- _ 下划线 → underscore
    
- ~ 波浪线 → tilde
    
- @ 电子邮件里的@ → at sign / at symbol
    
- # 井号 → hash / number sign / pound sign
    
    （社交媒体里一般说 hashtag）
    
- & 和号 → ampersand
    
- * 星号 → asterisk
        
- ^ 插入符号 / 抑扬符 → caret
    
- % 百分号 → percent sign
    
- ‰ 千分号 → per mille sign
    
- + 加号 → plus sign
        
- - 减号 → minus sign
        
- = 等号 → equals sign
    
- × 乘号 → multiplication sign / times sign
    
- ÷ 除号 → division sign
    
- < 小于号 → less-than sign
    
- > 大于号 → greater-than sign
    
- ≤ 小于等于号 → less-than or equal to sign
    
- ≥ 大于等于号 → greater-than or equal to sign
    
- ≠ 不等号 → not equal sign
    
- ° 度数符号 → degree symbol
    
- °C 摄氏度符号 → degrees Celsius
    
- $ 美元符号 → dollar sign
    
- € 欧元符号 → euro sign
    
- ¥ 日元/人民币符号 → yen sign / yuan sign
    
- § 节号（法律条文里的“第§条”）→ section sign
    
- ¶ 段落标记 → pilcrow / paragraph mark
    
- ′ 单撇 → prime (常用于角度的分，分钟)
    
- ″ 双撇 → double prime (角度的秒，秒数)
