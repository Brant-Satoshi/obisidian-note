- **Mac**：`⌥ + ⌘ + L`（Option + Command + L）
    
## **2. 控制台自动格式化设置**

如果你希望在 **运行 SQL 前自动格式化**：

1. 打开 **Preferences**（Mac：`⌘ + ,`）
    
2. 搜索 **SQL Dialects**，确保你当前连接的数据库类型正确（如 MySQL）
    
3. 搜索 **Code Style → SQL**
    
4. 在这里可以配置：
    
    - **Case**（关键字大小写，比如全部大写 `SELECT`）
        
    - **Alignment**（对齐方式）
        
    - **Wrapping**（换行规则）
        
5. 设置好后，格式化快捷键就会按照你的规则来排版。
