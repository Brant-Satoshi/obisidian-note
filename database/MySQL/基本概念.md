## 客户端连接

  ## **1. 本机终端连接（命令行方式）**

适合你直接在 Mac 上连接本地 MySQL。

bash

复制编辑

`mysql -u root -p`

- `-u root`：用户名是 root
    
- `-p`：提示输入密码（第一次安装可能无密码，直接按回车）
    
- 如果需要指定端口（默认 3306）：
    

bash

复制编辑

`mysql -u root -p -h 127.0.0.1 -P 3306`

---

## **2. 远程连接 MySQL**

如果你要从 Mac 连接另一台服务器上的 MySQL，需要：

- 服务器的 **IP 地址**
    
- MySQL 用户名和密码
    
- 远程访问权限（MySQL 默认只允许 localhost 连接，需要修改 `user` 表或 `bind-address`）
    

命令示例：

`mysql -u 用户名 -p -h 服务器IP -P 3306`

例如：

`mysql -u admin -p -h 192.168.1.100 -P 3306`
