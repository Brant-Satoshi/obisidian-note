| 命令                   | 作用          | 例子                   |
| -------------------- | ----------- | -------------------- |
| `\l` / `\list`       | 列出所有数据库     | `\l`                 |
| `\c dbname`          | 切换数据库       | `\c mydb`            |
| `\dt`                | 查看当前库里的表    | `\dt`                |
| `\d table`           | 查看表结构       | `\d users`           |
| `\di`                | 查看索引        | `\di`                |
| `\dv`                | 查看视图        | `\dv`                |
| `\df`                | 查看函数        | `\df`                |
| `\du`                | 查看角色（用户）    | `\du`                |
| `\dn`                | 查看所有 schema | `\dn`                |
| `\conninfo`          | 查看当前连接信息    | `\conninfo`          |
| `\password username` | 修改用户密码      | `\password postgres` |
| `\q`                 | 退出 psql     | `\q`                 |

---
## 连接
psql -U 用户名 -d 数据库名
psql -h 主机 -p 端口 -U 用户名 -d 数据库名

---

## 退出 psql
\q
