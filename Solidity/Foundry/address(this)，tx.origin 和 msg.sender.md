#  常见的address(this)，tx.origin 和 msg.sender 语句解释

- address(this)：当前合约的地址
- tx.origin：合约调用源地址
- msg.sender：合约的调用者

![[Pasted image 20250504040533.png]]
EOA 地址指的是 **Externally Owned Account（外部拥有账户）** 的地址，在以太坊中，EOA 是一种**由用户控制的账户类型**，与之对应的是 **合约账户（Contract Account）**。
## EOA 的特性

- 由 **公钥/私钥对**控制（如通过助记词生成）。
    
- 可以：
    
    - 发起交易（如转账、调用合约）
        
    - 支付 gas 费
        
- 本身**不包含代码**，只是一个地址和余额。

|类型|控制方式|能否发交易|有无代码|常见例子|
|---|---|---|---|---|
|**EOA**|用户私钥（签名）|✅ 可以|❌ 没有|MetaMask 钱包地址|
|**合约账户**|部署代码控制|❌ 被动执行|✅ 有|智能合约地址|
