### Forge (https://getfoundry.sh/introduction/overview#forge)

Master the core smart contract development tool with the [Forge overview](https://getfoundry.sh/forge/overview), covering building, testing, deploying, and verifying contracts.

### 

Cast[](https://getfoundry.sh/introduction/overview#cast)

Learn to interact with blockchain networks from the command line using [Cast](https://getfoundry.sh/cast/overview) for contract calls, transactions, and chain data retrieval.

### 

Anvil[](https://getfoundry.sh/introduction/overview#anvil)

Set up local development networks with [Anvil](https://getfoundry.sh/anvil/overview), Foundry's fast Ethereum-compatible node with forking capabilities.

### 

Chisel[
](https://getfoundry.sh/introduction/overview#chisel)

Explore Solidity interactively with [Chisel](https://getfoundry.sh/chisel/overview), the integrated REPL for rapid prototyping and debugging.
